# Week 3 - AI Career Recommendation System

## Description

A Python-based recommendation system that suggests suitable career paths based on user-entered technical skills. The system compares user skills with predefined role requirements and recommends the most relevant career options.

## Features

- Skill-based recommendation engine
- User input processing
- Career path matching
- Relevance score calculation
- Multiple career recommendations
- Interactive command-line interface

## Technologies Used

- Python
- Dictionaries
- Lists
- String Processing
- Recommendation Logic

## Workflow

1. Accept user skills as input
2. Process and clean skill data
3. Compare skills with predefined role requirements
4. Calculate matching skill scores
5. Sort recommendations by relevance
6. Display recommended career paths

## Results

- Interactive recommendation system
- Multiple career suggestions
- Ranked recommendations based on skill matching
- User-friendly command-line interface

## Files

- `recommendation.py` - Career recommendation system
- `Week3_Output.png` - Program execution output

## Learning Outcomes

- Recommendation system fundamentals
- Skill matching algorithms
- Data structures in Python
- Building intelligent decision-making systems
