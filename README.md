### Week 2 - Data Classification Using AI

#### Description

A Machine Learning classification project that uses the Iris Dataset and the K-Nearest Neighbors (KNN) algorithm to classify flower species based on their features.

#### Technologies Used

* Python
* Scikit-learn
* Machine Learning
* K-Nearest Neighbors (KNN)

#### Workflow

1. Load the Iris Dataset
2. Split data into training and testing sets
3. Apply feature scaling using StandardScaler
4. Train the KNN classifier
5. Generate predictions
6. Evaluate model performance

#### Results

* Dataset: Iris Dataset
* Algorithm: K-Nearest Neighbors (KNN)
* Accuracy Achieved: **100%**
* Confusion Matrix Generated
* Classification Report Generated

#### Files

* `classification.py`
* `Week2_Output.png`

### Week 3 - AI Career Recommendation System

#### Description

A Python-based recommendation system that suggests suitable career paths based on a user's technical skills. The system analyzes entered skills, compares them with predefined role requirements, and recommends the most relevant career options.

#### Technologies Used

* Python
* Dictionaries
* Lists
* String Processing
* Recommendation Logic

#### Workflow

1. Accept user skills as input
2. Process and clean skill data
3. Compare skills with predefined role requirements
4. Calculate matching skill scores
5. Rank recommendations based on relevance
6. Display recommended career paths

#### Results

* Interactive skill-based recommendation system
* Multiple career path suggestions
* Ranked recommendations based on matching skills
* User-friendly command-line interface

#### Files

* `recommendation.py`
* `Week3_Output.png`
