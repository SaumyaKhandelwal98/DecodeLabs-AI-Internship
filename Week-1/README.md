# Week 1 - Rule-Based AI Chatbot

## Description

A Python-based chatbot that responds to predefined user queries using a dictionary-based response system. This project demonstrates the fundamentals of conversational AI and rule-based decision making.

## Features

- Rule-based response system
- Dictionary-driven conversation handling
- Continuous interaction loop
- Input sanitization using string methods
- Fallback responses for unknown queries
- User-friendly command-line interface

## Technologies Used

- Python

## Files

- `chatbot.py` - Main chatbot application
- `Output.png` - Program execution output

## Learning Outcomes

- Understanding chatbot fundamentals
- Working with dictionaries in Python
- Handling user input and validation
- Building interactive command-line applications
